# Contributing to Lightning Maintenance

Thank you for considering contributing!

### How to contribute

- Report bugs or request features by opening an [Issue](https://github.com/devmusehq/lightning-maintenance/issues)
- Fork the repo and submit a Pull Request for improvements
- Keep changes small and focused (this plugin aims to stay lightweight)

### Code standards

- Follow WordPress coding standards
- Keep the plugin as lightweight as possible
- Test thoroughly with caching plugins enabled

Made with ❤️ by the DevMuseHQ team.
